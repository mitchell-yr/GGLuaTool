## 简介
这是一个悠然的GGlua脚本工具。用于编写、~编译~(遇到无法解决技术问题取消)、加密适用于GameGuardian的lua脚本的工具。
此软件目前只有部分功能，将在近期加入更多功能。正在持续更新！

此安卓软件使用了结绳编程4.0(tieLang)来制作。结绳编程是一种一中文编程安卓软件的语言，[结绳官方网站](http://tiecode.cn/)。

## 使用方法
> - 下载结绳编程，下载tsp文件，在结绳编程中导入即可
> - 下载结绳编程，将绳包(.tsp)导入

- 脚本编辑器
